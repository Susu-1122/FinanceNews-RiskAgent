# FinanceNews-RiskAgent

财经新闻风险联动智能调研助手。

本项目目标是构建一个 AI Agent，用于自动采集指定股票或行业的财经新闻，调用大模型进行多维度情感与风险评分，并结合 XGBoost 模型输出短期风险预测和中文调研报告。

## 项目目标

用户输入股票名称、股票代码或行业关键词后，系统自动完成：

1. 采集相关新闻
2. 清洗和整理新闻
3. 对新闻进行情绪与风险打分
4. 聚合新闻评分特征
5. 调用 XGBoost 模型预测风险
6. 生成中文调研报告

## 当前阶段

当前处于 V0 版本：

- 使用 Mock 新闻数据
- 使用规则进行简单情绪和风险评分
- 生成基础调研报告
- 暂不接入真实新闻源
- 暂不接入真实大模型
- 暂不训练真实 XGBoost 模型
